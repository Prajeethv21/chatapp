# Real Time Chat App

This is my internship task project built using React + TypeScript + Vite.

## Features implemented

- User sign-in screen (basic profile input)
- Chat room and one-to-one chat view
- Instant message sending
- Typing indicator for active conversation
- Back button to return to sign-in screen
- Responsive UI for desktop and mobile

## Run project

```bash
npm install
npm run dev
```

## Build and lint

```bash
npm run build
npm run lint
```

## Notes

- Current chat data is stored in local state for demo purpose.
- This can be connected to Socket.IO / backend API in next step.
